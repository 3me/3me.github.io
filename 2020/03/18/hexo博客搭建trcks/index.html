tricks 之文章插入图片

​ 当我们写 markdown 文章的时候，为了后期可移植性考虑，我们希望插入图片的时候用相对路径。那么当这篇文章需要发布到博客时，该怎么做呢？很简单，只需要在hexo 的[^1]配置文件 _config.yml 做如下设置。

​ 将 _config.yml 文件中的配置项 post_asset_folder 设为 true 后，执行命令 $ hexo new post name，在source/_posts中会生成文章 name.md和同名文件夹name。将图片资源放在name中，文章就可以使用相对路径引用图片资源了。[^2]

_posts/name/image.jpg

[^1]: 注意这里是 hexo 的 _config.yml 配置文件，而不是主题的 _config.yml配置文件，不要搞错了。
[^2]: 注意当写图片相对路径的时候，直接写图片名称 image.jpg即可，不需要再加上目录名 name。
